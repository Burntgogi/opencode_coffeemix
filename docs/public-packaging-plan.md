# Public Packaging Plan — opencode_coffeemix_all_0_1

**Workspace:** `opencode_coffeemix_all_0_1`
**Purpose:** classify every top-level file and directory for public release readiness.

---

## Classification key

| Label | Meaning |
|---|---|
| **keep public** | Core artifact suitable for external publication as-is |
| **archive** | Internal history or planning doc; move to `archive/` or remove from public tree |
| **exclude** | Local-only helper, generated output, or scratch; do not include in public package |
